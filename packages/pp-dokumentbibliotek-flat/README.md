# Dokumentbibliotek – flat visning

Prosjekttillegg for Prosjektportalen 365 som legger til en flat dokumentvisning
(uten mapper) i biblioteket **Dokumenter**, og inkluderer kolonnen **Bane** som
viser mappestrukturen til hvert dokument.

| | |
| --- | --- |
| **ID** | `pp-dokumentbibliotek-flat` |
| **Type** | extension |
| **Minimum PP-versjon** | 1.10.0 |
| **Tagger** | dokumenter, visning, tillegg |

> Generalisert prosjekttillegg hentet fra `prosjektportalen365-addons/Prosjekttillegg`.
