# Risikobibliotek

Innholdspakke (`type: content`) for Prosjektportalen 365 med et standard bibliotek
av vanlige prosjektrisikoer og risikoreduserende tiltak. Brukes som utgangspunkt
for risikovurdering i nye prosjekter.

| | |
| --- | --- |
| **ID** | `pp-risikobibliotek` |
| **Type** | content |
| **Minimum PP-versjon** | 1.10.0 |
| **Tagger** | risiko, kvalitet, innhold |

> Mock-pakke for utvikling/test av malpakkekatalogen.
