# Endringslogg - Veiprosjekt

Alle viktige endringer i denne malpakken dokumenteres i denne filen.

Formatet er basert på [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
og denne pakken følger [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-06-09

### Lagt til

- Første versjon av prosjektmalen for veg- og samferdselsprosjekter (faseinndeling, sjekklister og oppfølging av anleggsleveranser).
- Grunnstruktur og metadata på plass; provisjoneringsoppsettet (lister, innholdstyper og kolonner) fylles ut i senere versjoner.
