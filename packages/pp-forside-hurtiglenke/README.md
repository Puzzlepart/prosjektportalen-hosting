# Forside med hurtiglenker

Prosjekttillegg for Prosjektportalen 365 som legger til **hurtiglenke-webdelen**
på prosjektets forside (`ProjectHome.aspx`). Tilpass tillegget med dine egne lenker.

| | |
| --- | --- |
| **ID** | `pp-forside-hurtiglenke` |
| **Type** | extension |
| **Minimum PP-versjon** | 1.10.0 |
| **Tagger** | forside, navigasjon, tillegg |

> Generalisert prosjekttillegg hentet fra `prosjektportalen365-addons/Prosjekttillegg`.
