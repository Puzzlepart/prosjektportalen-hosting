# Fasesider

Prosjekttillegg for Prosjektportalen 365 som legger til unike prosjektforsider
for hver prosjektfase: **Konsept**, **Planlegge**, **Gjennomføre**, **Avslutte**
og **Realisere**. Tilpass tillegget med dine egne faser og innhold.

| | |
| --- | --- |
| **ID** | `pp-fasesider` |
| **Type** | extension |
| **Minimum PP-versjon** | 1.10.0 |
| **Tagger** | faser, forside, tillegg |

> Generalisert prosjekttillegg hentet fra `prosjektportalen365-addons/Prosjekttillegg`.
