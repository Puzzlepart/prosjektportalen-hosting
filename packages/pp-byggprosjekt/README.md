# Byggprosjekt

Prosjektmal tilpasset bygg- og anleggsprosjekter i Prosjektportalen 365.

Inneholder taksonomi for byggfaser, fasesjekkliste for prosjektering/utførelse,
dokumentmaler for entreprise og en valgfri venstremeny tilpasset byggfasene.

| | |
| --- | --- |
| **ID** | `pp-byggprosjekt` |
| **Type** | template |
| **Minimum PP-versjon** | 1.10.0 |
| **Tagger** | bygg, anlegg, prosjektledelse |

> Mock-pakke for utvikling/test av malpakkekatalogen.
